# Multi-Language Online IDE

## Overview

The Multi-Language Online IDE is a modern web application that enables users to write, compile, and execute code in multiple programming languages from a single platform. It provides a VS Code-like coding experience with syntax highlighting, code execution, custom input/output, and project management through a clean and responsive interface.

## Features

* Supports multiple programming languages (C, C++, Java, Python, JavaScript, TypeScript, Go, Rust, C#, PHP, and more)
* Monaco Editor with syntax highlighting and auto-completion
* Compile and run code instantly
* Custom input and output console
* Error and execution status display
* Create, save, rename, and delete projects
* Download and upload source code files
* Multiple editor themes
* Responsive and modern UI
* User authentication and project storage

## Technologies Used

* React.js
* Tailwind CSS
* Node.js
* Express.js
* MongoDB
* Monaco Editor
* Judge0 API / Piston API
* JWT Authentication

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project

```bash
cd online-ide
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
